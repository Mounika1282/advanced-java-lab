2. What are the main important features of Applets in Java? 
Applets run in a sandboxed environment, which restricts their access to system resources such as 
memory, processing power, and storage. Applets typically provide limited user interaction, usually 
through HTML controls or UI components provided by the applet itself. 
 
3. What are the key features of Java Servlets? 
• Servlets work on the server side. 
• Servlets are capable of handling complex requests obtained from the web server. 
• Generate dynamic responses efficiently.

6. What is the main role of JDBC and ODBC in database connectivity? 
JDBC is Java-specific and designed for Java applications to connect with relational databases. 
ODBC is language-independent and enables applications in various languages to connect with 
databases seamlessly.
